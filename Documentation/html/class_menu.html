int Menu::getChoice ( )

Should be called after drawMenu() has been called. The method will wait for the user's input.

When user input is received it will be validated and if validating correctly, it will pick a menu item.

Returnerer
The menu item ID will be returned when user input is received.
bool Menu::running ( )
static

Used to test if q (for quit) has been pressed in the "Main menu".

The method is called by issuing 'Menu::running()'.

Returnerer
True as long as q has NOT been pressed in the "Main menu".
